You are a Senior Staff Software Engineer, Performance Engineer, UX Engineer, and QA Lead.

Your task is to perform a complete audit and optimization of this entire website.

Objective

The website must feel premium, instant, smooth, and trustworthy on mobile and desktop.

Users should never experience:

* Janky animations
* Layout shifts
* Blank loading states
* Frozen UI
* Delayed button responses
* Slow page transitions
* Laggy scrolling
* Inconsistent behavior
* Broken business logic
* Race conditions
* State synchronization issues

Performance Audit

Analyze and optimize:

Core Web Vitals

* Largest Contentful Paint (LCP)
* Interaction to Next Paint (INP)
* Cumulative Layout Shift (CLS)

Rendering

* Unnecessary re-renders
* React state inefficiencies
* Expensive component updates
* Memory leaks
* Infinite render loops
* Unoptimized effects
* Excessive DOM nodes

Assets

* Images
* Videos
* Fonts
* SVGs
* Icons

Ensure:

* Proper lazy loading
* Responsive image sizing
* Modern formats
* Preloading critical assets
* Code splitting
* Tree shaking

Loading Experience

If any action takes longer than 100ms:

Replace waiting with perceived performance improvements.

Examples:

* Skeleton loaders
* Shimmer effects
* Progressive image loading
* Smooth fade-ins
* Optimistic UI updates
* Placeholder states
* Animated transitions

Users should always feel that something is happening.

Never show:

* Empty white screens
* Blank containers
* Frozen buttons
* Sudden content jumps

Review every page and user flow.

Check:

Homepage

Account Flow

* Friction
* Confusion

Reduce steps wherever possible.

Fix any discovered issues.
