✅ Simplified System (No tokens, no new DB tables)
💡 Core idea
Use Shopify order → product handle(s)
Build a direct product review link
Pass minimal data in query params (or even just product handle)
Reuse existing /products/[handle] page + review modal
1. Trigger (keep ONLY this logic)
Replace cron + tokens with:
A single cron job (or background worker) every 15 min:
SELECT * FROM paymob_intents
WHERE shipment_status = 'delivered'
AND review_email_sent_at IS NULL
AND delivered_at <= now() - interval '12 hours'

Then:

send email
set review_email_sent_at = now()

👉 That’s it. No token system.
